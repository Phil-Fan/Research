## 📝 使用说明

### 如何使用这个笔记系统

1. **从研究方向入手**
   - 点击左侧的研究方向（如 LLM/安全）
   - 阅读方向概要，了解研究背景
   - 查看重点论文推荐

2. **深入阅读论文**
   - 在方向页面中找到感兴趣的论文
   - 点击进入查看详细笔记
   - 参考论文的思维导图和讨论点

3. **持续更新**
   - 论文笔记会持续更新
   - 方向总结会定期维护
   - 欢迎提出建议和反馈
