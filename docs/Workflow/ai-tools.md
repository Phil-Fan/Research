# AI 辅助工具

## Claude Code

强大的 AI 编程助手，支持：

- 代码生成与重构
- 文档编写
- 调试与优化
- MCP (Model Context Protocol) 扩展

- Context7 MCP

## 文献阅读辅助

### Elicit

[Elicit](https://elicit.com/) - 基于自然语言的论文检索工具

- 用自然语言描述研究问题
- 自动检索相关论文
- 提取关键信息

### Semantic Scholar

[Semantic Scholar](https://www.semanticscholar.org/) - AI 驱动的研究工具

- AI 驱动的论文推荐
- 自动提取论文关键信息
- 引用关系分析

### Connected Papers

[Connected Papers](https://www.connectedpapers.com/) - 论文知识图谱

- 可视化论文之间的关系
- 发现相关研究
- 免费版每月 5 张图谱

## AI 辅助工作流建议

### 论文阅读

1. **快速筛选** - 使用 Elicit 检索相关论文
2. **关系梳理** - 使用 Connected Papers 了解论文关系
3. **深度阅读** - 使用 AI 辅助理解复杂概念
4. **笔记整理** - 使用 Zotero + Better Notes 记录

### 代码实现

1. **理解算法** - 使用 AI 解释复杂算法
2. **生成框架** - 使用 Claude Code 生成代码框架
3. **调试优化** - 使用 AI 辅助调试
4. **文档编写** - 使用 AI 生成文档注释

### 论文写作

1. **大纲设计** - 使用 AI 辅助设计论文结构
2. **语言润色** - 使用 Grammarly 检查语法
3. **翻译辅助** - 使用 DeepL 进行高质量翻译
