# 写作工具

## LaTeX

$\LaTeX$ 是学术写作的标准工具，特别适合包含大量数学公式的论文。

### 推荐资源

- [Overleaf](https://www.overleaf.com/) - 在线 LaTeX 编辑器
- [LaTeX Studio](https://www.latexstudio.net/) - 国内 LaTeX 资源

## 公式识别

### SimpleTex

[SimpleTex](https://simpletex.cn) - 强大的公式识别工具

- 支持图片中的公式识别
- 自动转换为 LaTeX 代码
- 支持批量识别

### 使用场景

- 从 PDF 中提取公式
- 图片公式转 LaTeX
- 手写公式识别

## 作图工具

### Excalidraw

[Excalidraw](https://excalidraw.com) - 开源的手绘风格绘图工具

**优点**:

- 类手绘风格，自然亲切
- 支持 CSV 格式自动画表格
- 可以由 Mermaid 导入
- 有 VSCode 和 MkDocs 插件

**适用场景**:

- 流程图
- 架构图
- 示意图

### Eraser

[Eraser](https://app.eraser.io)

**优点**:

- 自带一些图标
- 有 AI 辅助绘制功能（付费）
- 网页即可使用

**适用场景**:

- 快速绘制简单图表
- AI 辅助绘图

### PowerPoint / Keynote

**优点**:

- 自定义程度高
- 上手简单，上限很高
- 可以画出很精美的图

**适用场景**:

- 会议展示图
- 复杂示意图
- 需要精美排版的图

### Draw.io

[Draw.io](https://app.diagrams.net/)

**优点**:

- 有应用端也有网页端
- 设置较为丰富
- 完全免费

**适用场景**:

- 流程图
- 网络拓扑图
- 组织结构图

### Adobe Illustrator

[Adobe Illustrator](https://www.adobe.com/products/illustrator.html)

**优点**:

- 专业级矢量绘图工具
- 功能强大

**缺点**:

- 学习成本比较高

**适用场景**:

- 专业绘图
- 需要精确控制的插图
