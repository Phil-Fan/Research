## 工作流建议

### 论文阅读工作流

1. **搜集** - 通过 arXiv、Connected Papers 等平台搜集相关论文
2. **筛选** - 阅读标题和摘要，筛选相关文献
3. **管理** - 使用 Zotero 管理文献，添加标签和笔记
4. **精读** - 使用 Better Notes 模板做笔记
5. **总结** - 使用 Xmind 等工具整理思维导图

### 论文写作工作流

1. **文献整理** - Zotero 管理参考文献
2. **大纲设计** - 使用 Excalidraw 等工具画图
3. **正文撰写** - LaTeX 写作
4. **公式编辑** - SimpleTex 公式识别
5. **格式调整** - 使用 Zotero 样式生成参考文献

## 工具选择原则

1. **简单易用** - 学习成本低，上手快
2. **开放兼容** - 支持多种格式，易于迁移
3. **可扩展** - 支持插件，功能可定制
4. **跨平台** - 支持多操作系统
